PS5 Linux Loader is an open-source payload that exploits a patched hypervisor (HV) vulnerability to boot Linux bare-metal on PS5 Phat consoles. Once running, Linux has full access to the console's hardware — you can play Steam games, run emulators, browse the web, and use the PS5 as a capable desktop PC.

<Note>
  PS5 Linux Loader only supports **PS5 Phat** consoles on **firmware 3.xx or 4.xx**. PS5 Slim is not supported. Firmware 5.xx and above are not currently supported.
</Note>

## Hardware overview

The PS5 Phat hardware exposed to Linux after the HV exploit:

| Component | Specification |
|-----------|---------------|
| CPU | 8 cores / 16 threads at 3.5 GHz |
| GPU | AMD at 2.23 GHz (amdgpu driver) |
| Video output | HDMI — 1080p, 1440p, 2160p at 60 Hz |
| Storage | USB drive (required); M.2 SSD (optional, 4.xx only) |

## Architecture overview

PS5 Linux Loader works by exploiting a patched vulnerability in the PS5's hypervisor. The payload runs from userland after the jailbreak exploit gains code execution, then defeats the HV across all 16 CPU cores by patching Virtual Machine Control Blocks (VMCBs) via the IOMMU. With the hypervisor neutralized, the loader maps the Linux kernel and bootloader into memory and triggers a rest-mode cycle — when the PS5 wakes, it boots Linux instead of PS5 OS.

The approach gives Linux bare-metal access to hardware: no virtual machine, no sandboxing. This is why performance is close to native and why features like direct GPU access with amdgpu are possible.

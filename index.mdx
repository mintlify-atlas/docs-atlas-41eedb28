PS5 Linux Loader is an open-source payload that exploits a patched hypervisor (HV) vulnerability to boot Linux directly on PS5 Phat consoles running firmware 3.xx or 4.xx. Once booted, Linux runs bare-metal with full access to the console's hardware: 8 CPU cores at 3.5 GHz, an AMD GPU at 2.23 GHz, 4K60 HDMI output, all USB ports, and optional M.2 SSD storage.

## How it works

<Steps>
  <Step title="Build a Linux image">
    Clone the [ps5-linux-image](https://github.com/ps5-linux/ps5-linux-image) repository and run the build script to produce a bootable Ubuntu image, then flash it to a USB drive.
  </Step>
  <Step title="Configure your PS5">
    Enable **Supply Power to USB Ports** in Rest Mode settings and disable HDMI Device Link so the payload can trigger a controlled rest-mode boot.
  </Step>
  <Step title="Run the jailbreak exploit">
    Use [umtx2](https://github.com/idlesauce/umtx2) with a fake DNS server to serve the exploit through the PS5's built-in browser, giving userland code execution on the PS5.
  </Step>
  <Step title="Send the loader payload">
    Compile `ps5-linux-loader.elf` or download a release binary, then send it to the PS5 over TCP port 9021. The loader defeats the hypervisor, maps Linux files, and puts the PS5 into rest mode to boot Linux on wake.
  </Step>
</Steps>

<Note>
  PS5 Linux Loader is only supported on **PS5 Phat** consoles. PS5 Slim is not supported. Only firmware versions 3.xx and 4.xx are currently supported.
</Note>
